# WebSage
AI-powered web scraping tool with semantic search and data visualization using Python, Streamlit, and Gemini API.

## ✨ Features
- Extracts product data using BeautifulSoup & Requests
- Multi-language semantic search using Gemini API
- Natural language query support
- Data visualization with Matplotlib, Seaborn, and Plotly
- Interactive UI built with Streamlit

  ## 🛠 Tech Stack
- Python
- BeautifulSoup
- Requests
- Gemini API
- Streamlit
- Matplotlib, Seaborn, Plotly

  ## 📊 Usage
- Enter product link and query
- System scrapes data from websites
- Ask questions in natural language
- View insights through visual dashboards

- Add real-time scraping with APIs
